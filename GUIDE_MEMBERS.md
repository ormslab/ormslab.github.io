# ORMS Lab 홈페이지 수정 가이드 (구성원용)

코드를 몰라도 됩니다. 블로그 글 쓰듯이 홈페이지 내용을 수정할 수 있어요.

## 접속

1. 브라우저에서 **https://사이트주소/admin** 접속
2. **Login with GitHub** 클릭 (GitHub 계정 필요 — 관리자에게 초대를 요청하세요)

## 할 수 있는 것

| 메뉴 | 내용 | 예시 |
|---|---|---|
| News (소식) | 수상, 학회 참석, 공지 등 | "2026 춘계학술대회 발표" |
| Gallery (사진첩) | 사진 한 장 + 제목 + 날짜 | 회식, 야유회, 학회 사진 |
| Courses (강의) | 담당 강의 정보 | |
| Members (구성원) | 신입생 추가, 졸업생 → Alumni 변경 | |
| Publications (논문/특허) | 새 논문·특허 등록 | |
| Projects (연구과제) | 과제 추가, 종료 시 상태 변경 | |

## 글 쓰는 법 (예: News)

1. 왼쪽 메뉴에서 **News** 클릭 → **New News post**
2. 제목, 날짜, (선택) 사진, 내용 입력
3. 오른쪽 위 **Save** → 상태를 **Publish** (또는 Publish now) 클릭
4. **1~2분 뒤** 홈페이지에 자동 반영됩니다 (새로고침 필요)

## 자주 묻는 질문

- **저장했는데 사이트에 안 보여요** → 반영에 1~2분 걸립니다. 조금 뒤 새로고침하세요.
- **사진이 너무 커요** → 업로드 전에 폰에서 적당한 크기(2MB 이하 권장)로 줄여주세요.
- **잘못 올렸어요** → 해당 글을 열어 수정하거나 Delete 하면 됩니다.
- **멤버 사진 규격은?** → 정방형에 가까운 얼굴 사진이면 자동으로 원형으로 표시됩니다.

문제가 생기면 관리자(박지환)에게 연락하세요.
